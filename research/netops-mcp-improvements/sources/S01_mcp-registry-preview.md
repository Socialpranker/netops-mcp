# Introducing the MCP Registry

- Официальный реестр MCP запущен в preview 8 сентября 2025 г. на registry.modelcontextprotocol.io
- Работает как "открытый каталог и API для публично доступных MCP-серверов" — upstream-источник для всех downstream-клиентов и маркетплейсов
- Поддерживает и публичные sub-registries (opinionated marketplaces), и приватные enterprise-реестры
- Добавить сервер: следовать гайду "Adding Servers to the MCP Registry" на GitHub проекта
- Выгода от листинга: улучшенная discoverability, community moderation, видимость в downstream-каталогах которые тянут данные из официального реестра
- Для netops-mcp: листинг в официальном реестре даёт downstream-охват всех инструментов, строящихся поверх него
