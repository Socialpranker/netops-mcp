# Discovery Channels: Данные по эффективности

**Discovery источники с размерами:**
- PulseMCP: 11840+ серверов, hand-reviewed daily (quality signal)
- Smithery: 7000+ серверов, app-store интерфейс
- Glama: 21000+ серверов, largest volume
- MCP.so: 19700+ серверов, community-submitted
- mcp.directory: 3000+ с IDE integration
- mcpservers.org: 450+ curated (GitHub-native)
- mcpmarket.com: daily rankings by engagement (leaderboard)

**Three-tier discovery hierarchy:**
1. Quality-seekers → PulseMCP (hand-reviewed)
2. Maximum coverage → Glama или MCP.so
3. Developer/client builders → Official registry API

**Что делает листинг заметным:**
- Hand-curation (PulseMCP daily review)
- Install readiness (Smithery one-click)
- Security documentation
- Implementation detail (integration guides)
- Community ratings

**Критическое предупреждение:** "Trending is not fit" — leaderboards bias toward developer tools и не должны быть единственным proxy для релевантности.

Для netops-mcp: PulseMCP listing особенно ценен (hand-reviewed, quality signal) + Smithery для instant-install опыта.
