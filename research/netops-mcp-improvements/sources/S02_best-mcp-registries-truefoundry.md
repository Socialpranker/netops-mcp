# Best MCP Registries in 2026

- Ключевые реестры: Smithery (7000+ серверов, CLI-first, managed auth), Glama (21000+ серверов, meta-registry), MCP Market (10000+), MCP.so (19000+ серверов), TrueFoundry (enterprise)
- На май 2026: 27432 MCP-сервера агрегировано через три основных реестра (Official, Glama, Smithery)
- MCP Server Cards — стандарт для .well-known URL, позволяет каталогам автоматически индексировать сервер без подключения
- Smithery: один клик установка, handles OAuth flows автоматически; Glama: favors production-quality + clear docs; MCP.so: "any individual can add by submitting issue on GitHub", самый широкий охват
- Submission MCP.so: form-based. Submission Glama: manual review form. Submission awesome-mcp-servers: PR с правильным alphabetical placement
- Для netops-mcp: позиционирование в категории Monitoring + Cloud Platforms + Developer Tools увеличивает видимость в разных поисковых сценариях
